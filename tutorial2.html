How many parts of computer?
A computer has many parts, but the five main components are:

No.01--Motherboard:
The primary circuit board that holds all the hardware in place.

No.02--Central Processing Unit (CPU):
The "brain" of your computer.

No.03--Graphical Processing Unit (GPU):
Responsible for handling graphical operations and advanced imagery.

No.04--Random Access Memory (RAM):
Keeps important currently-used data and codes on standby.

No.05--Storage device:
Provides the memory for installing programs and saving files.
